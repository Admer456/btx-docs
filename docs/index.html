virtual PluginLibrary *LoadPluginLibrary(Path library) = 0

Reads plugin metadata and loads a plugin library.

One plugin library may contain multiple plugin implementations.

Example usage:

PluginLibrary* myPluginLibrary = PluginSystem->LoadPluginLibrary( "assets/MyPlugin" );

Parameters

libraryPath – directory of the metadata file

Returns

nullptr if the path is not valid or if it’s missing needed files, otherwise returns a valid PluginLibrary.

virtual const PluginLibraryMetadata &GetPluginLibraryMetadata(const IPlugin *plugin) const = 0

Gets plugin library info.

Returns

virtual void UnloadPluginLibrary(const PluginLibrary *pluginLibrary) = 0

template<typename pluginType = IPlugin>
inline void ForEachPluginOfType(std::function<void(pluginType*)> function)

Utility that exectues something for each plugin that implements pluginType (e.g. IApplication).

Example usage: 

// Go through each plugin instance that implements IApplication and update it
myPluginLibrary->ForEachPluginOfType<IApplication>( []( IApplication* app ) 
    {
        // Beginners: note that this is a lambda function
        app->Update();
    } );

virtual PluginList &GetPluginList(const char *interfaceName) = 0

Gets a list of plugins that implement interfaceName.

Example usage: 

for ( auto& plugin : PluginSystem->GetPluginList( "IApplication" ) )
{
    Console->Print( adm::format( "Plugin: %s", plugin->GetName() ) );
}
